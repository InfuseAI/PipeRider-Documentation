# Installation

### Requirements

{% hint style="info" %}
Python 3.8+
{% endhint %}

### Install or update the CLI

```shell
pip install -U piperider-cli
```

### Verify the version

```shell
piperider-cli version
```

### List all of commands

```shell
piperider-cli --help
```
